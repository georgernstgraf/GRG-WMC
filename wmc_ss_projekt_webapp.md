# Angabe zum individuellen Projekt WMC (Sommersemester)

Erstellen Sie eine kleine Webapplikation mit html, CSS und JavaScript. Wählen
Sie ein interessantes individuelles Thema.


## Anforderungen

- Semantisches html
- Styling mit CSS (Frameworks erlaubt, KI-Unterstützung erlaubt)
- Mindestens drei Interaktionen, davon mindestens eine Ereignisbehandlung und
  eine DOM-Modifikation
- Javascript: Strukturierung gemäß Vorlage mit 8 Punkten
- Verwendung eines State-Objektes
- Speichern des States in Local Storage
- Keine KI-Nutzung bei der Implementierung

## Optionale Erweiterungen

- Komplexere Programmlogik
- Aufruf von Web-APIs (z.B: `https://github.com/public-apis/public-apis`)
- Verwendung von Promises, async/await
- KI-Nutzung ist hierbei erlaubt/sinnvoll. Achten Sie jedoch auf eine saubere
  Abgrenzung. Zum Beispiel können Sie diese Erweiterungen als zweite Ausbaustufe
  in einem eigenen Ordner abgeben. 


# Beurteilung

- Abgabe bis Ende Mai via Git
- Eine positive Beurteilung erfolgt nur, wenn Sie zur Präsentation bereit sind.
- Alle Stunden im Juni haben somit prüfungsimmanenten Charakter, Anwesenheit ist unbedingt erforderlich.
- Bei der Präsentation müssen Sie den gesamten Code erklären, Fragen beantworten
  und kleine Änderungen vornehmen können.
- Primäres Beurteilungskriterium ist Funktionalität und sauberer Code, nicht Design.
- Besser weniger Umfang und gute Erklärungen, als komplexe Lösung die nicht gut
  erklärt werden kann.
